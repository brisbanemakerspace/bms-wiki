# How to make changes to this wiki
Good on you! Thanks for wanting to make BMS better by contributing to our wiki. The good news is, everyone with an active membership has automatic access to most wiki functions. You can already create and edit pages, upload photos etc.

## Make a change
1. Login using the profile button at the top of this page. We recommend using a desktop browser as it's a bit clunky on mobile.
2. After you login using your member portal account, you'll see a new "Page Actions" menu next to the profile icon. This allows you to make changes to certain pages.
3. To edit a page, navigate to it, then click on "Page Actions" in the top right, and select "Edit" to open the editor.
4. Click the "Save" button in the top right when you are done.

## Tips
### Don't rename page URLs
Please do not rename existing page URLs as you can accidentally break existing links and QR codes. If you have recently created a page and need to rename it that it is fine. However, do not rename existing pages please.

### QR codes should use the URL shortener
We do not recommend ever using the direct wiki page link when creating QR codes or printing a URL onto something physical like a sign or label. Ask someone on Discord to create you a short URL. This means your QR code is smaller/simpler and means we can update it remotely if the link ever changes for some reason.

### Red Links
Red links are links that point to a page that doesn't exist. This is most likely because that page hasn't been created yet, but it was flagged as one we should create. Please click on it then tap the "create page" button to create a new page.

If the page should already exist, you may need to update the link it points to. Find the correct page using the search bar, then edit the page as normal to update the link.

The final reason that can cause red links, is if a placeholder link was created to a non-existent page, but that page was later created. You will need to edit the page with the red link and re-save it so that it renders a new version and updates the cache.

### Restrictions
Some parts of the wiki however, are restricted for either legal, compliance or risk management purposes. Only certain members can make changes to the policy (which are legally enforceable by-laws) and infrastructure pages. If you need to make changes to these sections, please ask on Discord.
