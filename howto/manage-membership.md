# How to become a member?
Memberships are all managed through [the portal](https://portal.brisbanemaker.space).

See also: https://wiki.brisbanemaker.space/bylaws/membership-processes

These steps are for signing up through the new user experience. If you're a returning member or had clicked "Skip Signup (if you just want an account)", scroll down to the "Membership from existing user" section.
